发表于 2017-06-27   |     |   阅读次数

在编写 Android 程序时，为了在代码中控制 xml 文件中定义的组件，我们需要通过 findViewById() 这个方法来获取到对应组件。当一个界面中含有大量组件时，往往会产生许多冗杂的代码，且需要对组件进行批量控制时，也很不方便。

Butter Knife 是一个意图帮助编程人员解决 Android 界面绑定的框架，通过注解的方式帮助减少代码量，并在编译时生成原生代码，实现安全、简单。
