滚吧！ .tk

发表于 2016-10-16 更新于 2018-08-12 分类于 网站 阅读次数： 阅读次数： Valine：

曾经本博客用的也是 .tk 的免费域名，毕竟是个免费域名。直到后来买了腾讯云的一块钱 .cn 域名之后，才换了一个在一般人眼中看起来稍微正常点的域名。原来的 .tk 还曾经被老外吐槽过。。。

当时一直不明白为什么.tk会被认为是含 malicious 内容的不安全网站。直到最近接了一个网站项目……

客户要求做一个网站，没有域名要求，本来是打算直接 ip 访问的，但是手上正好那么多免费的.tk域名，干脆随便用一个吧。

上线之后，一些用户（约15%）反映无法访问网站(っﾟДﾟ)っ!!。因为网站是用 react 写的，吾辈首先想到的是浏览器的兼容性问题。

然而，，，结果怎样呢？吾辈放几张图大家感受下：

这是.tk的 ping 情况：

没有延迟数据的地区情况大致如下：

总结一下，大多数电信线路无法解析.tk域名，问题明显与DNS有关。同样是托管在 dnspod 上，.cn却安然无恙。也就是说，.tk可能被电信认为是有害的网站，所以没有解析，这也许就是免费域名的弊端吧。

果然天下是没有免费的午餐的，不过电信这 dns 做得有点绝啊。这次的 bug 通过更换域名算是解决了，姑且为还在用免费域名的站长提个醒吧。网站追求的更多是稳定，一个.cn域名一年也就几十块钱，还不到服务器一个月的租金，有条件的还是不要用免费啦。
